# Veora 

### Descripción de la aplicación
Veora es una plataforma interactiva que recomienda una película o serie diaria a los usuarios. Al finalizar la visualización y escribir una reseña, el usuario gana puntos que puede donar para apoyar causas solidarias relacionadas con el Hambre Cero, Agua Limpia y Acción por el Clima.

### Tecnologías utilizadas
Flutter y Dart para una interfaz multiplataforma e intuitiva
Firebase para la base de datos en la nube (Firestore) y autenticación
The Movie Database (TMDB) para el catálogo de películas y series
GitHub para el repositorio del código

### Estructura de carpetas
La estructura organizada del proyecto es la siguiente
/lib/assets: Archivos multimedia como el logo de Veora
/lib/widgets: Componentes reutilizables de la interfaz (botones, tarjetas)
/lib/screens: Pantallas principales (Inicio, Recomendación Diaria, Donaciones, Perfil)
/lib/styles: Definición de la estética visual

### Instrucciones básicas para ejecutar el proyecto
Se debe tener instalado Flutter 
Eecutar flutter pub get en la terminal para descargar las dependencias necesarias y usar el repositorio del proyecto para ejecutarlo
Conectar un emulador o dispositivo físico y ejecutar el comando flutter run.
Se puede ejecutar desde Google, pulsando F12 y pulsando "CTRL + SHIFT + M" para verlo como un móvil.
